# Sales Management Web App

A simple and efficient web application to manage products, sales, stock, and business analytics.

## Features

- User authentication (localStorage-based)
- Product management (add / delete products)
- Sales recording system
- Analytics dashboard:
- Total revenue (CA)
- Revenue per day
- Top selling products
- History filtering (Today / All)
- Visual charts (sales performance)
- Export sales data to CSV
- Stock reset functionality
- Real-time stock updates

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)

## Purpose

This project was built to practice:

- Business logic implementation
- DOM manipulation
- State management with localStorage
- Building a complete frontend application

## Limitations

- No backend (data stored in localStorage)
- No real authentication security
- Single-device usage

## Possible Improvements

- Backend integration (Django / Node.js)
- Secure authentication (JWT, sessions)
- Database (PostgreSQL / MongoDB)
- Multi-user real-time sync
- Deployment (cloud hosting)
