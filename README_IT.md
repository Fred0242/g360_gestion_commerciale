# Applicazione di gestione vendite

Un'applicazione web semplice per gestire prodotti, vendite, stock e statistiche.

## Funzionalità

- Autenticazione utente (basata su localStorage)
- Gestione prodotti (aggiunta / eliminazione)
- Registrazione delle vendite
- Dashboard:
- Fatturato totale
- Fatturato giornaliero
- Prodotti più venduti
- Filtro storico (Oggi / Tutto)
- Grafici delle prestazioni
- Esportazione dati in CSV
- Reset dello stock
- Aggiornamento stock in tempo reale

## Tecnologie

- HTML5
- CSS3
- JavaScript (Vanilla)

## Obiettivo

Questo progetto è stato realizzato per:

- Implementare logica di business
- Manipolare il DOM
- Gestire dati lato client
- Costruire un'applicazione frontend completa

## Limitazioni

- Nessun backend (solo localStorage)
- Autenticazione non sicura
- Utilizzo su un solo dispositivo

## Miglioramenti possibili

- Backend (Django / Node.js)
- Autenticazione sicura
- Database
- Sincronizzazione multi-utente
- Deploy online
